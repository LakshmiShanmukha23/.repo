Selectors

  • Element Selector

    • EXAMPLE:
    p { font-size:25px; } here p is element selector.

  • Class Selector

    • class selector denoted with period(.).








    EXAMPLE:
    .big { font-size:25px; } here .big is class selector.

  • ID Selector

    • ID selector denoted with #.






    EXAMPLE:
    #big { font-size:25px; } here #big is ID selector.

  • Grouping Selector

    • EXAMPLE:

    p,div { font-size:25px; }

    here p,div are different elements selectors,there grouping gives grouping selectors.







  • Combining Selector

    1. Element With Class Selector

      2. EXAMPLE:

      p.big { font-size:15px; }













      here p is element selector and .big is class selector there combination gives elent with class selector.

    2. Child Selector

      3. EXAMPLE:
      article>p { font-size:25px; }


      here p is element selector and article is a tag there combination gives child selector.
      It tells p is direct child of article.which means the property satisfies for only p which is direct child of article.

    3. Descendent Selector

      4. EXAMPLE:
      article p { font-size:25px; }


      here p is element selector and article is a tag there combination gives descendent selector.
      It tells p is child of an article.which means the property satisfies for all p which is child of article.

  • Not Limited Element Selector

    • EXAMPLE:
    .big p { font-size:25px; }

    here p is element selector and .big is a class selector there combination gives not limited to element selector.
    It tells p with class big.which means the property satisfies for only p with class big(p class="big").

    EXAMPLE 2:

    article>.big { font-size:25px; }






    here .big is class selector and article is a tag there combination gives not limited to element selector.
    It tells .big is direct child of article.which means the property satisfies for only .big which is direct child of article.

  • Pseudo Class Selector
    1. :link
    2. :visited
    3. :hover
    4. :active
    5. :nth-child
Style Placements
  • Inline styling

    • EXAMPLE:
    p style="font-size:5px;"
    It is Inline styling. It should write on body tag.It does not need style tags.

  • Internal styling

    • EXAMPLE:
    p { style=font-size:5px; }
    It is Internal styling. It should write on head tag.It need style tags.

  • External styling

    • EXAMPLE:
    p { style=font-size:5px; }
    It is External styling. It should write on head tag.It doesnot need style tags. It need refferal(link) or path of that stylesheet.
    FOR EXAMPLE
    link rel="stylesheet" href="styles.css"
